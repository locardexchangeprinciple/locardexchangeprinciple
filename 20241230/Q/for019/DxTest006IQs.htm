Network Forensics Quiz

1. What is the primary tool used for packet analysis in network forensics?
Wireshark
2. What is the main purpose of intrusion detection systems (IDS) in network forensics?
To detect malicious activity or security breaches on a network
3. Which protocol is commonly analyzed in network forensics to monitor web traffic?
HTTP
4. What is the term for the log data generated by network devices that help in forensic investigations?
Network Logs
5. In network forensics, what does 'packet sniffing' refer to?
The process of capturing and analyzing data packets on a network
6. What is the role of a firewall in network forensics?
To filter incoming and outgoing network traffic based on security rules
7. What type of data is commonly used in network forensics to identify the source and destination of network traffic?
IP Addresses
8. Which protocol is primarily used to transmit email data, often analyzed in network forensics?
SMTP
9. What is the term used to describe the process of recreating the events leading to a network security incident?
Timeline Reconstruction
10. What is a common challenge faced by network forensics investigators when analyzing encrypted network traffic?
The difficulty of decrypting the traffic without the necessary keys
