Recovering Deleted and Hidden Data Quiz

1. What is the primary method for recovering deleted files from a storage device?
File Carving
2. What is the term for the space on a disk that is marked as unused but may still contain remnants of deleted files?
Slack Space
3. What type of file system allows the recovery of deleted files by marking them as deleted, but not overwriting the data immediately?
FAT32
4. What is a hidden file that is not normally visible to users, but can be detected through forensic methods?
Steganographic Files
5. Which tool is commonly used to recover deleted files or hidden data?
FTK Imager
6. What is the term for the process of searching for and extracting data from the unallocated space of a disk?
Data Carving
7. What is the name for the small fragments of data left behind in unallocated space after a file is deleted?
File Fragments
8. Which file system type is less likely to leave traces of deleted files due to overwriting data quickly?
NTFS
9. What is the most important factor in determining whether deleted data can be recovered?
Overwriting of the Data
10. What is the term used for files that are deliberately hidden within other files, like images or audio?
Steganography
